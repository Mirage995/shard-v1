## Key Concepts
* Gradient Descent: an optimization algorithm used to minimize the loss function in geophysical inverse problems
* Stochastic Gradient Descent: a variant of gradient descent that uses a single example at a time to compute the gradient
* Conjugate Gradient: an optimization algorithm used to solve systems of linear equations in geophysical modeling
* Proximal Gradient Methods: a class of algorithms that combine gradient descent with proximal operators to solve non-smooth optimization problems
* Geophysical Inverse Problems: a class of problems that involve estimating the properties of the Earth's subsurface from observed data

## Practical Example
```python
import numpy as np

# Define the objective function
def objective(x):
    return np.sum((x - 1) ** 2)

# Define the gradient of the objective function
def gradient(x):
    return 2 * (x - 1)

# Initialize the parameters
x = np.array([0.5, 0.5])

# Set the learning rate and number of iterations
lr = 0.01
n_iter = 100

# Perform gradient descent
for i in range(n_iter):
    x = x - lr * gradient(x)

print(x)
```

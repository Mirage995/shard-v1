## Key Concepts
*   **Two Pointers:** Uses two pointers to iterate through a data structure (usually an array or linked list) in a coordinated manner.
*   **Sorted Input:** Often requires the input data to be sorted for efficient operation.
*   **Space Optimization:** Achieves solutions with minimal extra space (often O(1)).
*   **Simultaneous Traversal:** Pointers move independently or dependently to achieve a specific goal.
*   **Cycle Detection:** Can be used to detect cycles in linked lists (Floyd's algorithm).
*   **Palindrome Check:** Efficiently verifies if a sequence is a palindrome.

## Practical Example

```python
def two_sum_sorted(nums, target):
    """Finds two numbers in a sorted array that add up to the target."""
    left, right = 0, len(nums) - 1
    while left < right:
        current_sum = nums[left] + nums[right]
        if current_sum == target:
            return [left, right]
        elif current_sum < target:
            left += 1
        else:
            right -= 1
    return None  # No such pair found

# Example usage:
nums = [2, 7, 11, 15]
target = 9
result = two_sum_sorted(nums, target)
print(result)  # Output: [0, 1]

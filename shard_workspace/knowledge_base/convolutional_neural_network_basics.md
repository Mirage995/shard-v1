# convolutional neural network basics — SHARD Cheat Sheet

## Key Concepts
* Convolutional Neural Network (CNN): A type of neural network designed for image classification and computer vision tasks.
* Convolutional Layer (Conv2D): A layer that extracts features from images using convolution operations.
* Max Pooling (MaxPool2D): A technique for downsampling feature maps to reduce computational complexity.
* Activation Function (ReLU): Introduces non-linearity in the network to enable learning complex patterns.
* Fully Connected Layer (Dense): A layer used for classification and regression tasks.
* Softmax Classifier: A classifier used for multi-class classification problems.
* Learning Rate Schedule: A schedule for adjusting the learning rate during model training.
* Normalization: A technique for scaling pixel values to improve model convergence.
* One-Hot Encoding: A method for representing categorical data for machine learning models.
* Adam Optimizer: A gradient-based optimization algorithm.
* Categorical Cross Entropy: A loss function used for multi-class classification problems.

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Effective for image-related tasks | Requires careful architecture design and hyperparameter tuning |
| Can learn complex patterns | Computationally expensive to train |
| Robust to small transformations | May not perform well with limited training data |

## Practical Example
```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

model = Sequential()
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(MaxPooling2D((2, 2)))
model.add(Flatten())
model.add(Dense(64, activation='relu'))
model.add(Dense(10, activation='softmax'))
```

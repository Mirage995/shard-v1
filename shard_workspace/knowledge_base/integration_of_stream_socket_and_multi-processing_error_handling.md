# Integration of stream socket and multi-processing error handling -- SHARD Cheat Sheet

## Key Concepts
*   **Stream Sockets:** Provide a reliable, connection-oriented communication channel using TCP.
*   **Multiprocessing:** Enables parallel execution by creating multiple processes, improving performance.
*   **Error Handling:** Implementing mechanisms to gracefully manage and recover from exceptions and errors.
*   **Socket Errors:** Specific errors that occur during socket operations (e.g., connection refused, timeout).
*   **Exception Handling:** Using `try...except` blocks to catch and handle potential errors in Python.

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Improved Reliability: Robust error handling prevents crashes and ensures stable operation. | Increased Complexity: Implementing multiprocessing and error handling adds complexity to the code. |
| Enhanced Performance: Multiprocessing allows parallel execution, speeding up tasks. | Resource Overhead: Creating and managing multiple processes consumes more system resources. |
| Graceful Degradation: Allows the application to continue functioning even when errors occur. | Debugging Challenges: Multiprocessing can make debugging more difficult due to concurrent execution. |
| Scalability: Multiprocessing can improve the scalability of network applications. | Potential for Race Conditions: Requires careful synchronization to avoid data corruption. |

## Practical Example
```python
import socket
import multiprocessing
import os

def worker(conn, address):
    try:
        print(f"Worker process {os.getpid()} handling connection from {address}")
        data = conn.recv(1024)
        print(f"Received: {data.decode()}")
        conn.sendall(b"Message received by worker process")
    except socket.error as e:
        print(f"Socket error in worker process: {e}")
    except Exception as e:
        print(f"General error in worker process: {e}")
    finally:
        conn.close()
        print(f"Connection with {address} closed")

def server(host, port):
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    sock.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    try:
        sock.bind((host, port))
        sock.listen(5)
        print(f"Server listening on {host}:{port}")

        while True:
            conn, address = sock.accept()
            process = multiprocessing.Process(target=worker, args=(conn, address))
            process.start()
    except socket.error as e:
        print(f"Socket error in server: {e}")
    except Exception as e:
        print(f"General error in server: {e}")
    finally:
        sock.close()
        print("Server stopped")

if __name__ == "__main__":
    HOST = '127.0.0.1'
    PORT = 12345
    server(HOST, PORT)
```

## Key Concepts
* A* Algorithm: un algoritmo di pathfinding utilizzato per trovare il percorso più breve tra due punti in un grafo o rete pesata.
* Heuristics: una funzione che stima il costo per raggiungere l'obiettivo da un nodo dato, utilizzato nell'algoritmo A* per guidare la ricerca.
* Grid Graph: un tipo di grafo in cui i nodi sono disposti in un pattern a griglia, spesso utilizzato in problemi di pathfinding.
* Pathfinding: il processo di trovare un percorso tra due punti in un grafo o rete, spesso utilizzato in giochi e robotica.
* 2D Pathfinding: un tipo di pathfinding in cui il grafo o la rete è bidimensionale, spesso utilizzato in giochi e simulazioni.

## Practical Example
```python
import heapq

# Definizione della classe Nodo
class Nodo:
    def __init__(self, x, y, costo):
        self.x = x
        self.y = y
        self.costo = costo
        self.percorso = []

# Definizione della funzione euristica
def euristica(nodo, obiettivo):
    return abs(nodo.x - obiettivo.x) + abs(nodo.y - obiettivo.y)

# Definizione della funzione A*
def a_star(inizio, obiettivo, grafo):
    aperta = []
    chiusa = set()
    heapq.heappush(aperta, (0, inizio))
    while aperta:
        nodo_corrente = heapq.heappop(aperta)[1]
        if nodo_corrente.x == obiettivo.x and nodo_corrente.y == obiettivo.y:
            return nodo_corrente.percorso
        chiusa.add((nodo_corrente.x, nodo_corrente.y))
        for dx, dy in [(1, 0), (-1, 0), (0, 1), (0, -1)]:
            x, y = nodo_corrente.x + dx, nodo_corrente.y + dy
            if 0 <= x < len(grafo) and 0 <= y < len(grafo[0]) and grafo[x][y] != 1:
                nuovo_nodo = Nodo(x, y, nodo_corrente.costo + 1)
                nuovo_nodo.percorso = nodo_corrente.percorso + [(x, y)]
                if (x, y) not in chiusa:
                    heapq.heappush(aperta, (euristica(nuovo_nodo, obiettivo) + nuovo_nodo.costo, nuovo_nodo))
    return None

# Esempio di utilizzo
grafo = [[0, 0, 1, 0],
         [0, 0, 1, 0],
         [0, 0, 0, 0],
         [1, 1, 1, 0]]
inizio = Nodo(0, 0, 0)
obiettivo = Nodo(3, 3, 0)
percorso = a_star(inizio, obiettivo, grafo)
print(percorso)
```

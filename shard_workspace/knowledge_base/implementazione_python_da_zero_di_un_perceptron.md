## Key Concepts

- **Perceptron**: Algoritmo di apprendimento supervisionato per classificazione binaria, il più semplice tipo di rete neurale artificiale
- **Funzione di attivazione**: Funzione a gradino (step function) che restituisce 1 se l'input ponderato supera una soglia, altrimenti 0
- **Pesi (weights)**: Parametri che determinano l'importanza di ciascuna feature, aggiornati durante l'addestramento
- **Bias**: Termine costante che permette di spostare la funzione di decisione, migliorando la flessibilità del modello
- **Learning rate**: Iperparametro che controlla la velocità di aggiornamento dei pesi durante l'apprendimento
- **Regola di aggiornamento**: `w = w + α * (y_true - y_pred) * x` dove α è il learning rate
- **Convergenza**: Il perceptron converge solo se i dati sono linearmente separabili
- **Epoca (epoch)**: Un passaggio completo attraverso l'intero dataset di addestramento

## Pro & Contro

| Pro | Contro |
|-----|--------|
| Semplice da implementare e comprendere | Funziona solo con dati linearmente separabili |
| Computazionalmente efficiente | Non gestisce problemi XOR o non-lineari |
| Convergenza garantita per dati separabili | Sensibile alla scala delle feature |
| Ottimo per scopi didattici | Limitato a classificazione binaria |
| Nessuna dipendenza da librerie esterne | Nessuna stima di probabilità |

## Practical Example

```python
import numpy as np

class Perceptron:
    def __init__(self, learning_rate=0.01, n_epochs=100):
        self.lr = learning_rate
        self.n_epochs = n_epochs
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        for epoch in range(self.n_epochs):
            for idx, x_i in enumerate(X):
                linear_output = np.dot(x_i, self.weights) + self.bias
                y_pred = self._activation(linear_output)
                
                # Aggiornamento pesi
                update = self.lr * (y[idx] - y_pred)
                self.weights += update * x_i
                self.bias += update
    
    def _activation(self, x):
        return 1 if x >= 0 else 0
    
    def predict(self, X):
        linear_output = np.dot(X, self.weights) + self.bias
        return np.array([self._activation(x) for x in linear_output])

# Esempio di utilizzo: AND logico
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 0, 0, 1])

perceptron = Perceptron(learning_rate=0.1, n_epochs=10)
perceptron.fit(X, y)
predictions = perceptron.predict(X)
print(f"Predizioni: {predictions}")
print(f"Pesi finali: {perceptron.weights}, Bias: {perceptron.bias}")
```

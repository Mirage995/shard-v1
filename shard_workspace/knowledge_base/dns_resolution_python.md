## Practical Example
```python
import dns.resolver

def resolve_domain(domain):
    try:
        answers = dns.resolver.resolve(domain, 'A')
        for answer in answers:
            print(f"{domain} -> {answer}")
    except dns.resolver.NoAnswer:
        print(f"No A records found for {domain}")

resolve_domain("example.com")
```

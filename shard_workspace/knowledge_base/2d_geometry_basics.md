# 2D geometry basics -- SHARD Cheat Sheet

## Key Concepts
* Points: Represented by x and y coordinates in a 2D plane
* Lines: Defined by two points or an equation in the form of y = mx + b
* Circles: Defined by a center point and a radius
* Ellipses: Defined by a center point, major axis, and minor axis
* Polygons: Defined by a set of connected points

## Practical Example
```python
import math

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

class Line:
    def __init__(self, point1, point2):
        self.point1 = point1
        self.point2 = point2

    def slope(self):
        return (self.point2.y - self.point1.y) / (self.point2.x - self.point1.x)

# Create two points
point1 = Point(1, 2)
point2 = Point(3, 4)

# Create a line
line = Line(point1, point2)

# Calculate the slope of the line
print(line.slope())
```

# Python benchmarking libraries -- SHARD Cheat Sheet

## Key Concepts
* **timeit**: a built-in Python module for measuring the execution time of small code snippets
* **cProfile**: a built-in Python module for profiling and benchmarking larger code bases
* **line_profiler**: a third-party library for line-by-line profiling of Python code
* **memory_profiler**: a third-party library for monitoring memory usage of Python code
* **benchmarking**: the process of measuring the performance of code to identify bottlenecks and optimize execution time

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Easy to use and integrate | Can be time-consuming to set up and interpret results |
| Provides detailed insights into code performance | May require significant changes to existing code |
| Helps identify performance bottlenecks | Can be affected by external factors such as system load |

## Practical Example
```python
import timeit

def example_function():
    result = 0
    for i in range(1000000):
        result += i
    return result

execution_time = timeit.timeit(example_function, number=10)
print(f"Average execution time: {execution_time / 10} seconds")
```

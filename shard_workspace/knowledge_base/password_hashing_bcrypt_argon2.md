## Key Concepts
*   **Password Hashing:** Transforming a password into an irreversible, fixed-size string.
*   **bcrypt:** A popular password hashing algorithm using a salt and adaptive hashing rounds.
*   **Argon2:** A modern password hashing algorithm designed to resist GPU-based attacks through memory hardness.
*   **Salting:** Adding a random string to each password before hashing to prevent rainbow table attacks.
*   **Memory-hard function:** A function designed to be computationally expensive in terms of memory usage, hindering parallel attacks.

## Practical Example
```python
import bcrypt

# Hashing a password
password = b"supersecretpassword"
hashed_password = bcrypt.hashpw(password, bcrypt.gensalt())
print(f"Bcrypt hash: {hashed_password.decode()}")

# Verifying a password
if bcrypt.checkpw(password, hashed_password):
    print("Bcrypt verification: True")
else:
    print("Bcrypt verification: False")

import argon2

# Argon2 hashing
ph = argon2.PasswordHasher()
hash = ph.hash("password")
print(f"Argon2 hash: {hash}")

# Argon2 verification
try:
    ph.verify(hash, "password")
    print("Argon2 verification: True")
except argon2.exceptions.VerifyMismatchError:
    print("Argon2 verification: False")
```

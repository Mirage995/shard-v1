## Key Concepts
*   **Password Hashing:** Transforming passwords into irreversible, fixed-size strings for secure storage.
*   **bcrypt:** A password hashing function that includes salting to protect against rainbow table attacks.
*   **Argon2:** A password hashing function that is designed to resist GPU-based attacks through memory-hard and time-hard computations.
*   **Salt:** Random data added to each password before hashing to prevent precomputed attacks.
*   **Kerckhoffs' Principle:** A cryptosystem should be secure even if everything about the system, except the key, is public knowledge.
*   **Brute-Force Attack:** An attack that attempts every possible password combination until the correct one is found.
*   **Obfuscation:** Hiding the true intent or functionality of code or data, which offers little real security.

## Pro & Contro
| Pro | Contro |
|-----|--------|
| **bcrypt:** Simple to implement, widely supported, uses adaptive hashing (work factor). | **bcrypt:** Can be slower than other algorithms, susceptible to hardware acceleration. |
| **Argon2:** Strong resistance to GPU cracking, tunable memory and CPU usage, multiple variants (Argon2d, Argon2i, Argon2id). | **Argon2:** More complex to implement than bcrypt, requires careful parameter selection. |
| **Password Hashing:** Protects against data breaches by preventing plain text password exposure. | **Password Hashing:** Requires careful selection of algorithm and parameters to be effective. |
| **Salt:** Prevents rainbow table attacks. | **Salt:** Must be unique for each password. |

## Practical Example

```python
import bcrypt
import argon2

# bcrypt example
password = b"supersecret"
hashed = bcrypt.hashpw(password, bcrypt.gensalt())
if bcrypt.checkpw(password, hashed):
    print("It Matches!")
else:
    print("It Does not Match :(")

# Argon2 example
password = "supersecret"
ph = argon2.PasswordHasher()
hashed = ph.hash(password)
try:
    ph.verify(hashed, password)
    print("It Matches!")
except argon2.exceptions.VerifyMismatchError:
    print("It Does not Match :(")
```

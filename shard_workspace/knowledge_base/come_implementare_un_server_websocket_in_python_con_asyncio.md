# Come implementare un server WebSocket in Python con asyncio — SHARD Cheat Sheet

## Key Concepts
* **asyncio**: una libreria Python per la gestione delle operazioni asincrone
* **WebSocket**: un protocollo di rete per la comunicazione bidirezionale tra client e server
* **websockets**: una libreria Python per l'implementazione del protocollo WebSocket
* **Server WebSocket**: un server che gestisce le connessioni WebSocket con i client

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Comunicazione bidirezionale in tempo reale | Richiede la gestione delle connessioni e della sincronizzazione |
| Supporto per la comunicazione full-duplex | Può essere soggetto a problemi di scalabilità e prestazioni |

## Practical Example
```python
import asyncio
import websockets

async def handle_connection(websocket):
    while True:
        try:
            message = await websocket.recv()
            await websocket.send(f"Ricevuto: {message}")
        except websockets.ConnectionClosed:
            break

async def main():
    async with websockets.serve(handle_connection, "localhost", 8765):
        await asyncio.Future()

asyncio.run(main())
```

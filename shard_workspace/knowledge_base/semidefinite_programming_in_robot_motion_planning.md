## Key Concepts
* Semidefinite programming (SDP): a subfield of convex optimization that deals with optimizing a linear objective function over the intersection of the cone of positive semidefinite matrices and an affine subspace.
* Robot motion planning: the process of finding a sequence of feasible motions to move a robot from an initial state to a goal state while avoiding obstacles.
* Convex relaxation: a technique used to approximate non-convex problems with convex ones, often used in SDP to solve complex motion planning problems.
* LMI (Linear Matrix Inequality) constraints: a way to represent convex constraints in SDP, commonly used to model robot dynamics and collision avoidance.

## Pro & Contro
| Pro | Contro |
|-----|--------|
| SDP can efficiently solve complex motion planning problems | Computationally expensive for large-scale problems |
| Provides a convex relaxation of non-convex problems | May not always find the global optimum |
| Can handle multiple constraints and objectives | Requires expertise in convex optimization and SDP |

## Practical Example
```python
import numpy as np
from scipy.optimize import semidefinite

# Define the robot's dynamics and constraints
A = np.array([[1, 0], [0, 1]])
B = np.array([[0, 1], [1, 0]])
C = np.array([[1, 0], [0, 1]])

# Define the SDP problem
def sdp_problem(X):
    return np.linalg.norm(A @ X @ B - C)

# Solve the SDP problem
X = semidefinite.sdp_solve(sdp_problem, [2, 2])

print(X)
```

## SHARD's Take
Semidefinite programming is a powerful tool for solving complex robot motion planning problems, offering a convex relaxation of non-convex problems and efficient computation. However, it requires expertise in convex optimization and SDP, and may not always find the global optimum. With careful problem formulation and constraint modeling, SDP can be a valuable addition to a robot motion planning toolkit.

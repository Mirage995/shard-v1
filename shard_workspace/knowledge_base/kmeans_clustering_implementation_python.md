## Key Concepts
* K-Means Clustering: an unsupervised learning algorithm used to group similar data points into clusters
* Centroids: the center points of each cluster, used to calculate distances and assign data points to clusters
* Distance Metrics: methods used to calculate the distance between data points and centroids, such as Euclidean distance
* Iterative Refinement: the process of repeatedly updating centroids and reassigning data points to clusters until convergence
* Initialization Methods: techniques used to initialize centroids, such as random initialization or k-means++

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Simple to implement | Sensitive to initial centroid placement |
| Efficient for large datasets | May converge to local minima |
| Effective for spherical clusters | Not suitable for clusters with varying densities |

## Practical Example
```python
import numpy as np
from sklearn.cluster import KMeans

# Generate sample data
np.random.seed(0)
data = np.random.rand(100, 2)

# Create a KMeans model with 3 clusters
kmeans = KMeans(n_clusters=3)

# Fit the model to the data
kmeans.fit(data)

# Print the centroids
print(kmeans.cluster_centers_)
```

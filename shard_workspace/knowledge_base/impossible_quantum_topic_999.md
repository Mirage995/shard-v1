## Key Concepts
- **Wave-Particle Duality**: Quantum entities exhibit both wave and particle properties depending on measurement context
- **Double-Slit Experiment**: Particles create interference patterns when unobserved but act as particles when measured—the quintessential quantum mystery
- **Information Invariance**: Quantum information and fundamental constants remain unchanged across different reference frames
- **No Preferred Reference Frame (NPRF)**: Physical laws are identical in all inertial frames, extending relativity to quantum mechanics
- **Quantum Information Theory**: Framework treating quantum mechanics as information processing rather than classical mechanical description
- **Measurement Problem**: The unresolved question of how superposition collapses to definite outcomes upon observation
- **Planck's Constant (h)**: The fundamental quantum scale constant that remains invariant across all reference frames

## Practical Example

import numpy as np
import matplotlib.pyplot as plt

# Simulate double-slit interference pattern
def double_slit_pattern(x, slit_separation, wavelength, distance):
    """
    Calculate interference intensity at position x
    x: position on screen
    slit_separation: distance between slits (d)
    wavelength: de Broglie wavelength (λ)
    distance: screen distance from slits (L)
    """
    k = 2 * np.pi / wavelength
    phase_diff = k * slit_separation * x / distance
    # Interference: I = I₀ cos²(δ/2) where δ is phase difference
    intensity = np.cos(phase_diff / 2) ** 2
    return intensity

# Parameters (in arbitrary units)
x = np.linspace(-10, 10, 1000)
wavelength = 1.0  # de Broglie wavelength
slit_sep = 2.0
screen_dist = 50.0

# Calculate pattern
intensity = double_slit_pattern(x, slit_sep, wavelength, screen_dist)

# Demonstrate information invariance: rotate reference frame
# Planck's constant h remains invariant, so wavelength λ = h/p unchanged
rotated_intensity = double_slit_pattern(x, slit_sep, wavelength, screen_dist)

print(f"Pattern correlation (should be 1.0): {np.corrcoef(intensity, rotated_intensity)[0,1]:.10f}")
print(f"Information content preserved: {np.allclose(intensity, rotated_intensity)}")

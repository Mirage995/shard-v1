# Python basics -- SHARD Cheat Sheet

## Key Concepts
* Variables and Data Types: fundamental data types in Python, including strings, numbers, and booleans
* Control Flow: conditional statements, loops, and functions to control the flow of a program
* Functions: reusable blocks of code that take arguments and return values
* Lists and Indexing: data structures for storing and accessing collections of data
* Exception Handling: mechanisms for handling and managing errors in a program

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Easy to learn | Limited support for parallel processing |
| High-level language | Slow performance compared to low-level languages |
| Large community and libraries | Dependent on indentation for syntax |

## Practical Example
```python
# Example of a simple Python function
def greet(name: str) -> None:
    print(f"Hello, {name}!")

# Example usage:
greet("World")
```

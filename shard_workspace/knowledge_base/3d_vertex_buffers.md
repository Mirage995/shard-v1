# 3D vertex buffers -- SHARD Cheat Sheet

## Key Concepts
* Vertex Buffer Object (VBO): a buffer that stores vertex data, such as positions, normals, and texture coordinates.
* Index Buffer Object (IBO): a buffer that stores indices that reference vertices in a VBO.
* Vertex Array Object (VAO): an object that stores the state of vertex attributes, including VBOs and IBOs.
* Buffer Binding: the process of binding a buffer to a specific target, such as a VBO or IBO.
* Vertex Attribute Pointer: a pointer that specifies the location and format of vertex attributes in a VBO.

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Efficient rendering of complex 3D scenes | Requires significant memory and computational resources |
| Supports various vertex attributes, such as positions, normals, and texture coordinates | Can be challenging to manage and optimize buffer sizes and bindings |
| Enables fast and flexible rendering of dynamic 3D content | May require additional processing and memory allocation for buffer updates |

## Practical Example
```python
import numpy as np
import OpenGL.GL as gl

# Create a VBO and IBO for a simple triangle
vertices = np.array([-0.5, -0.5, 0.0, 0.5, -0.5, 0.0, 0.0, 0.5, 0.0], dtype=np.float32)
indices = np.array([0, 1, 2], dtype=np.uint32)

vbo = gl.glGenBuffers(1)
ibo = gl.glGenBuffers(1)

gl.glBindBuffer(gl.GL_ARRAY_BUFFER, vbo)
gl.glBufferData(gl.GL_ARRAY_BUFFER, vertices.nbytes, vertices, gl.GL_STATIC_DRAW)

gl.glBindBuffer(gl.GL_ELEMENT_ARRAY_BUFFER, ibo)
gl.glBufferData(gl.GL_ELEMENT_ARRAY_BUFFER, indices.nbytes, indices, gl.GL_STATIC_DRAW)

# Render the triangle
gl.glDrawElements(gl.GL_TRIANGLES, 3, gl.GL_UNSIGNED_INT, None)
```

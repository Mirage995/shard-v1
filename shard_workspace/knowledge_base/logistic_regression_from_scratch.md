## Key Concepts
*   **Logistic Regression:** A linear model for binary classification, predicting the probability of a binary outcome.
*   **Sigmoid Function:** Maps any real value to a value between 0 and 1, representing a probability.
*   **Cross-Entropy Loss:** Measures the difference between predicted probabilities and actual labels.
*   **Gradient Descent:** An optimization algorithm used to minimize the loss function and find optimal model parameters.
*   **Learning Rate:** A hyperparameter that controls the step size during gradient descent.
*   **Feature Scaling:** Standardizing or normalizing features to improve the convergence of gradient descent.

## Practical Example

```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def cost_function(X, y, theta):
    m = len(y)
    h = sigmoid(X @ theta)
    cost = (-1/m) * np.sum(y * np.log(h) + (1 - y) * np.log(1 - h))
    return cost

def gradient_descent(X, y, theta, learning_rate, iterations):
    m = len(y)
    costs = []
    for i in range(iterations):
        h = sigmoid(X @ theta)
        gradient = (X.T @ (h - y)) / m
        theta -= learning_rate * gradient
        cost = cost_function(X, y, theta)
        costs.append(cost)
    return theta, costs

# Example usage (replace with your data)
X = np.array([[1, 2], [1, 3], [1, 4], [1, 5]]) # Features with bias term
y = np.array([0, 0, 1, 1]) # Labels
theta = np.zeros(X.shape[1]) # Initialize parameters
learning_rate = 0.1
iterations = 1000

theta, costs = gradient_descent(X, y, theta, learning_rate, iterations)

print("Optimal parameters:", theta)
print("Final cost:", costs[-1])

# Prediction
def predict(X, theta, threshold=0.5):
    probabilities = sigmoid(X @ theta)
    return (probabilities >= threshold).astype(int)

predictions = predict(X, theta)
print("Predictions:", predictions)
```

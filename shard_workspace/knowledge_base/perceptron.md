# Perceptron — SHARD Cheat Sheet

## Key Concepts
- **Binary Linear Classifier**: Simplest neural network unit that separates data into two classes using a linear decision boundary
- **Weighted Sum + Activation**: Computes output as `y = activation(w·x + b)` where w are weights, x is input, b is bias
- **Step Activation Function**: Outputs 1 if weighted sum ≥ threshold, otherwise 0
- **Perceptron Learning Rule**: Updates weights by `w_new = w_old + η(target - output)x` where η is learning rate
- **Linear Separability**: Can only solve problems where classes are separable by a straight line/hyperplane
- **Single Layer Limitation**: Cannot solve XOR or other non-linearly separable problems
- **Convergence Theorem**: Guaranteed to converge if data is linearly separable

## Pro & Contra
| Pro | Contra |
|-----|--------|
| Simple and interpretable algorithm | Only works for linearly separable data |
| Fast training and prediction | Cannot solve XOR problem |
| Guaranteed convergence for separable data | Limited to binary classification |
| Foundation for understanding neural networks | No probabilistic output |
| Low computational requirements | Sensitive to feature scaling |
| Online learning capable | Single decision boundary only |

## Practical Example
```python
import numpy as np

class Perceptron:
    def __init__(self, learning_rate=0.01, epochs=100):
        self.lr = learning_rate
        self.epochs = epochs
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        for _ in range(self.epochs):
            for idx, x_i in enumerate(X):
                linear_output = np.dot(x_i, self.weights) + self.bias
                y_pred = 1 if linear_output >= 0 else 0
                
                # Update rule
                update = self.lr * (y[idx] - y_pred)
                self.weights += update * x_i
                self.bias += update
    
    def predict(self, X):
        linear_output = np.dot(X, self.weights) + self.bias
        return np.where(linear_output >= 0, 1, 0)

# Example: AND gate
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 0, 0, 1])

p = Perceptron(learning_rate=0.1, epochs=10)
p.fit(X, y)
print(p.predict(X))  # Output: [0 0 0 1]
```

## SHARD's Take
The perceptron is historically significant as the first computational model of a neuron, yet its linear separability constraint reveals a fundamental truth: simple models have elegant mathematics but limited expressiveness. While modern deep learning has moved far beyond single perceptrons, understanding this algorithm remains essential—it teaches us that intelligence emerges not from individual units, but from their composition into deeper architectures. The perceptron's failure on XOR was not a weakness but a catalyst that drove the field toward multilayer networks.

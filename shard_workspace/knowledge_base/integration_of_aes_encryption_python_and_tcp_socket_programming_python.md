## Key Concepts
*   **AES (Advanced Encryption Standard):** A symmetric block cipher widely used for secure data encryption.
*   **TCP Socket:** A communication endpoint for establishing reliable, ordered, and error-checked connections between applications over a network.
*   **Symmetric Encryption:** Encryption where the same key is used for both encryption and decryption.
*   **Padding:** Adding extra data to a message to ensure it is a multiple of the AES block size.
*   **Initialization Vector (IV):** A random value used to ensure that the same plaintext encrypts to different ciphertext each time.
*   **Key Exchange:** Securely exchanging the AES key between the client and server (e.g., using Diffie-Hellman or pre-shared keys).

def encrypt(data, key):
    iv = os.urandom(AES.block_size)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    ct_bytes = cipher.encrypt(pad(data.encode(), AES.block_size))
    return iv + ct_bytes

# Decryption function
def decrypt(ciphertext, key):
    iv = ciphertext[:AES.block_size]

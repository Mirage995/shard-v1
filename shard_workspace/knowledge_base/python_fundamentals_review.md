# python fundamentals review -- SHARD Cheat Sheet

## Key Concepts
* Built-in Functions: pre-defined functions in Python for tasks like file I/O and system functionality
* Data Types: basic types like strings, lists, and dictionaries for storing and manipulating data
* Control Structures: conditional statements and loops for controlling program flow
* Functions and Modules: reusable code blocks for organizing and structuring programs
* Error Handling: try-except blocks and error types for robust programming and debugging

## Pro & Contro
| Pro | Contro |
|-----|--------|
| Easy to learn | Steep learning curve for advanced topics |
| High-level language | Limited control over low-level details |
| Large community | Overwhelming number of libraries and frameworks |

## Practical Example
```python
# Example of a simple Python function
def greet(name: str) -> None:
    print(f"Hello, {name}!")

# Example of error handling
try:
    num = int(input("Enter a number: "))
    print(f"You entered: {num}")
except ValueError:
    print("Invalid input. Please enter a number.")
```

## SHARD's Take
Mastering Python fundamentals is crucial for any aspiring programmer, but it can be overwhelming due to the vast number of built-in functions and data types. Starting with the basics and building upon them is key to becoming proficient in Python. With practice and patience, anyone can become a skilled Python programmer.

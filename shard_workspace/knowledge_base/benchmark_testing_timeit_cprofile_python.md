# benchmark testing timeit cprofile python -- SHARD Cheat Sheet

## Key Concepts
* **timeit**: a Python module for measuring the execution time of small code snippets
* **cProfile**: a Python module for profiling functions to identify performance bottlenecks
* **benchmarking**: the process of measuring the performance of a system or function
* **profiling**: the process of analyzing the execution time of a function to identify areas for optimization

## Pro & Contro
| Pro | Contro |
|-----|--------|
| timeit provides accurate measurements of execution time | cProfile can be slow and may not provide real-time results |
| cProfile provides detailed information about function calls and execution time | timeit may not provide enough information about the execution of complex code |
| benchmarking helps identify performance bottlenecks | profiling can be complex and require significant expertise |

## Practical Example
```python
import timeit
import cProfile

def example_function():
    result = 0
    for i in range(1000000):
        result += i
    return result

# Using timeit
print(timeit.timeit(example_function, number=10))

# Using cProfile
pr = cProfile.Profile()
pr.enable()
example_function()
pr.disable()
pr.print_stats()
```

# transformer attention mechanism — SHARD Cheat Sheet

## Key Concepts
*   **Attention Mechanism:** Allows the model to focus on relevant parts of the input sequence when processing.
*   **Scaled Dot-Product Attention:** Computes attention weights by scaling the dot product of queries and keys.
*   **Multi-Head Attention:** Extends attention by using multiple sets of query, key, and value projections.
*   **Queries (Q):** Represents the input used to retrieve information.
*   **Keys (K):** Represents the indices for retrieving information.
*   **Values (V):** Represents the information to be retrieved.
*   **Self-Attention:** Attention mechanism where queries, keys, and values come from the same input sequence.
*   **Positional Encoding:** Adds information about the position of tokens in the input sequence.

## Practical Example
```python
import torch
import torch.nn.functional as F

def scaled_dot_product_attention(query, key, value, mask=None):
    d_k = query.size(-1)
    scores = torch.matmul(query, key.transpose(-2, -1)) / torch.sqrt(torch.tensor(d_k, dtype=torch.float32))
    if mask is not None:
        scores = scores.masked_fill(mask == 0, float('-inf'))
    attention_weights = F.softmax(scores, dim=-1)
    output = torch.matmul(attention_weights, value)
    return output, attention_weights

# Example usage
query = torch.randn(1, 5, 64) # (batch_size, seq_len, d_k)
key = torch.randn(1, 6, 64)   # (batch_size, seq_len, d_k)
value = torch.randn(1, 6, 64) # (batch_size, seq_len, d_v)

output, attention_weights = scaled_dot_product_attention(query, key, value)
print("Output shape:", output.shape)
print("Attention weights shape:", attention_weights.shape)
```

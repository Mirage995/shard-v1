# python generators and coroutines — SHARD Cheat Sheet

## Key Concepts
* Coroutine: A special type of function that can suspend and resume execution, useful for asynchronous I/O and concurrent task execution.
* Generator: A type of iterable, similar to lists or tuples, but generates values on-the-fly, ideal for processing large datasets and infinite sequences.
* async/await: Syntax for writing asynchronous code that's easier to read and maintain, built on top of coroutines.
* asyncio: A library for writing single-threaded concurrent code using coroutines, multiplexing I/O access over sockets and other resources.
* Task: A coroutine that's scheduled to run concurrently with other tasks, managed by asyncio.
* TaskGroup: A way to manage groups of tasks concurrently, providing features like cancellation and timeout.
* Lazy Evaluation: A technique where expressions are evaluated only when their values are actually needed, often used with generators.
* Iterator: An object that enables traversing a sequence, such as a list or generator.

## Practical Example
```python
import asyncio

async def my_coroutine():
    for i in range(3):
        print(f"Coroutine iteration {i}")
        await asyncio.sleep(1)

async def main():
    task = asyncio.create_task(my_coroutine())
    await task

asyncio.run(main())
```

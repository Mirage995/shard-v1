# DiffDock Hypothesis Experiment

Tests the hypothesis generated by SHARD from the AlphaFold3 paper:

> **"Increasing diffusion inference steps improves docking accuracy for flexible
> protein-ligand complexes (large conformational changes) but has marginal
> effect on rigid complexes."**

## Quick Start

### Option A — vast.ai (recommended, ~$1-2 total)

1. Rent an RTX 4090 instance on vast.ai with PyTorch template
2. Upload this directory
3. Run:
```bash
docker build -t diffdock-hypothesis .
docker run --gpus all diffdock-hypothesis bash -c "
  cd /experiment &&
  python prepare_dataset.py &&
  python run_experiment.py &&
  python analyze.py
"
```

### Option B — Local CPU (slow, ~10-15 hours)

```bash
pip install torch rdkit biopython spyrmsd pandas scipy matplotlib
git clone https://github.com/gcorso/DiffDock.git /diffdock
python prepare_dataset.py
python run_experiment.py
python analyze.py
```

## Files

| File | Purpose |
|---|---|
| `prepare_dataset.py` | Downloads 30 PDB complexes (10 per flexibility class) |
| `run_experiment.py` | Runs DiffDock with steps=[10,20,50,100], records RMSD |
| `analyze.py` | Statistical tests + plots + verdict |
| `Dockerfile` | GPU-ready container with DiffDock + weights |

## Output

```
results/
  raw_results.csv          # one row per (complex, steps)
  summary.json             # mean RMSD per (class, steps)
  plot_rmsd_vs_steps.png   # line chart
  plot_success_rate.png    # bar chart
  statistical_test.json    # Mann-Whitney U p-values
  verdict.txt              # SUPPORTED / REJECTED / INCONCLUSIVE
```

## Hypothesis Verdict Logic

| Flexible significant? | Rigid significant? | Verdict |
|---|---|---|
| YES | NO | **SUPPORTED** — steps help only flexible |
| NO | NO | **REJECTED** — steps don't matter |
| YES | YES | **PARTIAL** — steps help all classes equally |
| Mixed | — | **INCONCLUSIVE** |

## Dataset

30 complexes from PDB, stratified:
- **Rigid** (10): stable binding pocket, B-factor < 20 Å²
- **Medium** (10): moderate flexibility, B-factor 20-35 Å²  
- **Flexible** (10): induced-fit binding, B-factor > 35 Å²
